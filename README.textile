Provides a collection of useful commands for getting weather information over
IRC.

h2. Usage

| !weather _[zip]_ | Returns a textual description of the weather at a zip code. |
| !metar _[airport]_ | Displays the current weather at an airport (by ICAO code) in coded METAR format. |
| !taf _[airport]_ |  Displays the forecasted weather at an airport (by ICAO code) in coded TAF format. |
